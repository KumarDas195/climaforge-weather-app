# 🌦 ClimaForge - Advanced Weather App

ClimaForge is a modern responsive weather forecasting web application built using HTML, CSS, JavaScript, Tailwind CSS, and WeatherAPI.

It provides:
- Real-time weather updates
- 24-hour forecast
- 7-day forecast
- Air Quality Index (AQI)
- Humidity, UV Index, Wind, Visibility
- Responsive glassmorphism UI
- Mobile-friendly design

## ✨ Features

- 🌍 Search weather by city
- 📱 Fully responsive design
- 🌡 Real-time temperature
- 💨 Wind speed & direction
- 💧 Humidity tracking
- ☀ UV Index monitoring
- 🌫 Air Quality Index
- 📅 7-Day Forecast
- ⏰ 24-Hour Forecast
- 🎨 Modern glassmorphism UI


## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- Font Awesome
- WeatherAPI


## 🌐 API Used

WeatherAPI
